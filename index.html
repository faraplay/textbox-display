OFA Font & Textbox Displayer

Instructions

Copy text into the box below and click "Draw" to view the text in the OFA font.
You can also try clicking the "Parse and draw" button - this will attempt to parse the text entered into the box as a series of messages. Click here to go to the line breaker utility page.

Format for parsing:

  • All lines beginning with a # (hash symbol) are comments and will be ignored. You can use this to comment out the original Japanese text.
  • The first line should be unedited (a filename of the form commu2/par/***.par).
  • Next is a series of entries:
Each entry should have the following form:
  1. One line with a number, the ID of the text.
  2. One line with the name of the speaker (possibly blank, e.g. if the speaker is the Producer).
  3. One or more lines containing the actual text.
  4. One line with a single $ (dollar symbol) on it.
Make sure there are no blank lines after the last $ symbol.
